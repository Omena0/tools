## Proj

A tool to manage git repositories for your projects.

Usage: proj [mk|rm|mv|ls] [project name] <[new project name]>

### Examples

Create a project named `tools`
```sh
$ proj mk tools
```

List all projects
```sh
$ proj ls
```

Rename project
```sh
$ proj mv tools tools2
```

Delete the project
```sh
$ proj rm tools2
```
